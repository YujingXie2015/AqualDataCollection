About Us

Our goal is to build an Android mobile application system which helps to collect water leakage data from user’s client app. The functions we will implement includes collecting user’s information, receiving water leakage report from registered user, requesting nearby user’s input for water leakage report based on adaptive filtering algorithm.

Features

Water Leakage Report/

provide expert and quality services from detecting underground water leaks and repairs to supply pipe, water mains and water supply infrastructure installation

Real-time Location Detection/

Push notification based on user current device location to implement event tracking, instant & real-time data analysis functions

/Dynamic Dashboard Report

Display dashboard to view information in the database, and monitor events and trends of our application

/Embedded Google Map

Effective mapping service to provide accurate location for leakage report without manual input for map coordinates

Our applications would facilitate listening to water leakage report, organizing location­specified information, as well as sending notifications based on user’s location, time and credit.
